# 梦幻花园创作者审稿脚本

自动批量审核 Fanbook 花园创作者投稿，依据9条规则判断通过/拒绝，并自动提交审核结果。

---

## 第一次使用（只需做一次）

### 第一步：安装 Python
如果电脑没有 Python，先下载安装：
👉 https://www.python.org/downloads/

> ⚠️ 安装时必须勾选 **"Add Python to PATH"**

### 第二步：双击运行 `setup.bat`
会自动安装所需依赖，并引导你填写 API Key。

### 第三步：填写 API Key
用记事本打开 `config.json`，把 `api_key` 改成你的阿里云 DashScope Key：
```json
{
  "api_key": "sk-xxxxxxxxxxxxxxxx",
  "base_url": "https://coding.dashscope.aliyuncs.com/v1",
  "model": "qwen-vl-max"
}
```
> API Key 在阿里云控制台 → 模型服务 → 我的API-KEY 获取

---

## 每次审稿流程

**第一步：** 双击 `启动审稿浏览器.bat`，在打开的 Edge 里登录 Fanbook 审稿后台

**第二步：** 双击 `审稿.bat`，粘贴活动链接和主题，等待自动完成

就这两步。

---

## 活动链接在哪里找？

登录 Fanbook 审稿后台，点开要审核的活动，直接把浏览器地址栏的链接复制粘贴进去即可，脚本会自动识别活动ID。

---

## 审核规则说明

| 规则 | 说明 |
|------|------|
| 一稿多投 | 同一视频链接出现多次 |
| 视频时长 | 不足10秒不通过 |
| 无二创 | 纯原始游戏画面无任何创作元素 |
| 微博引流 | 微博投稿含非活动话题 |
| 内容相关 | 与梦幻花园游戏完全无关 |
| 纯数字文案 | 标题为3位以上纯数字 |
| 主题相关 | 视频内容与本期主题不符（日常活动宽松判断） |

> 默认审核标准偏宽松，有疑问的稿件一律通过。

---

## 出了问题怎么办？

**报错"用户未登录"** → Edge 里重新打开 Fanbook 审稿页面，刷新一下再跑

**报错"未找到 Fanbook 审稿标签"** → 确认用的是 `启动审稿浏览器.bat` 打开的 Edge，不是普通方式打开的

**某条稿件判断有误** → 在 Fanbook 后台手动改，脚本输出的 CSV 文件里有所有记录
